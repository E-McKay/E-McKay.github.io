Character Design - Robot boy


A game-ready character created to test out new software and techniques. This robot has many features designed to make it appear like a youth such as freckles, headphones and a slouched stance.




3D Game Assets - Planetside 2

3 Suits of cosmetic armour created for the game "Planetside 2" (DayBreak Games). Each suit of armour had to be created to fit on top of a base mesh, and had to be rigged, textures and modelled within the games aesthetic and technical constraints. Each suit also had to be created in 5 variants to fit with the games 'classes'.
